# SpringWebApp
Simple project, which was made to show how spring-framework-5 works. The projects stores a simple model, in which we can find entites such as: author, book and publisher.
# Technology Stack
* JDK 18.0.1 (Language Level Java 18),
* Apache Maven 4.0.0,
* Spring Boot 2.7.1,
* H2database 2.1.214,
